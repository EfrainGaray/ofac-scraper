🔐✍️ Generador de Request JWS + JWE para PAR

Herramienta para firmar Y encriptar peticiones PAR (JWE anidado con JWS)

⚠️ Información:

Esta herramienta genera un request FIRMADO y ENCRIPTADO (JWE anidado) para el flujo PAR.

Proceso: 1️⃣ Firma el payload con tu clave privada (JWS/PS256) → 2️⃣ Encripta el JWS con la clave pública de Keycloak (JWE/RSA-OAEP+A256GCM)

Nota: La clave privada NO se envía al servidor, todo el procesamiento es local en tu navegador.

0️⃣ Clave Privada del Cliente

Selecciona el archivo de clave privada en formato PEM

Formato PEM - Debe comenzar con -----BEGIN PRIVATE KEY----- o -----BEGIN RSA PRIVATE KEY-----

Identificador de la clave en el header JWT. Se usa para que el servidor sepa qué clave pública usar del JWKS para validar la firma.

1️⃣ Configuración de Keycloak

Ejemplo: keycloak-desarrollo.finanzasabiertas.cl

El realm de Keycloak (por defecto: cmf)

2️⃣ Datos del Cliente

Scopes separados por espacios

3️⃣ Parámetros PKCE

Valor aleatorio generado automáticamente (necesario para Token Exchange)

BASE64URL(SHA256(code_verifier))

4️⃣ Parámetros Adicionales

Identificador único del consentimiento

Objeto JSON con detalles de autorización (RAR). Dejar vacío si no se usa RAR
