# Pruebas pre-productivas en Sandbox (ATENA)

## **Introducci&oacute;n**

El objetivo de esta documentaci&oacute;n es describir el alcance funcional y modelo operativo del entorno Sandbox Tecnol&oacute;gico del Sistema de Finanzas Abiertas (SFA), habilitado para la ejecuci&oacute;n de pruebas t&eacute;cnicas entre participantes regulados y terceros autorizados, previo paso a entornos productivos.

## **Objetivos espec&iacute;ficos**

- 
Permitir validaci&oacute;n t&eacute;cnica de integraciones API bajo est&aacute;ndares SFA.

- 
Probar flujos de autenticaci&oacute;n y autorizaci&oacute;n (FAPI 2.0, OIDC, mTLS).

- 
Validar interoperabilidad entre actores (IPI, PSBI, IPC, PSIP).

- 
Ejecutar bater&iacute;as de pruebas estructuradas con evidencia trazable.

- 
Identificar brechas t&eacute;cnicas antes de ambientes productivos.
